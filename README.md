# 📝 Todo App

A modular Todo App built with **Vanilla JavaScript** and **Webpack** as part of [The Odin Project](https://www.theodinproject.com) curriculum.

---

## ✨ Features

- ✅ Create and delete projects
- 📌 Add, delete, and complete todos
- 📂 Multiple project support
- 🔀 Active project switching
- 🚦 Priority-based todos
- 💾 LocalStorage persistence
- 📱 Responsive modern UI

---

## 🛠️ Tech Stack

- **JavaScript** (ES6 Modules)
- **Webpack**
- **HTML5**
- **CSS3**
- **LocalStorage API**

## 🚀 Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/khushicomp/todo-app.git
cd todo-app
npm install
```

Run the development server:

```bash
npm start
```

Build for production:

```bash
npm run build
```
